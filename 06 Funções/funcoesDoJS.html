Funções já incluidas no JavaScript

alert(): exibe uma mensagem na caixa de diálogo;
confirm(): exibe uma mensagem na caixa de diálogo, com as opções "OK" e "Cancelar";
prompt(): exibe uma mensagem na caixa de diálogo, com um campo para entrada de dados;
Number(): converte o valor informado para o tipo numérico;
parseInt(): converte o valor informado para o tipo numérico inteiro;
parseFloat(): converte o valor informado para o tipo numérico decimal;
toFixed(): formata o valor numérico, definindo a quantidade de casas decimais;
replace(): substitui um caractere por outro caractere;
length: retorna a quantidade de caracteres de uma string;
toUpperCase(): converte todos os caracteres de uma string para maiúsculo;
toLowerCase(): converte todos os caracteres de uma string para minúsculo;
trim(): remove os espaços em branco no início e no final de uma string;
split(): divide uma string em um array de strings, de acordo com o caractere informado;
join(): junta os elementos de um array, separando-os pelo caractere informado;
sort(): ordena os elementos de um array;
reverse(): inverte a ordem dos elementos de um array;
push(): adiciona um elemento no final de um array;
pop(): remove o último elemento de um array;
shift(): remove o primeiro elemento de um array;
unshift(): adiciona um elemento no início de um array;
slice(): retorna um novo array, de acordo com o início e o fim informados;
splice(): remove ou substitui elementos de um array, de acordo com o início e a quantidade informados;
concat(): concatena dois ou mais arrays;
indexOf(): retorna a posição de um elemento de um array;
lastIndexOf(): retorna a última posição de um elemento de um array;
includes(): verifica se um elemento existe em um array;
filter(): retorna um novo array, de acordo com a condição informada;
map(): retorna um novo array, de acordo com a condição informada;
reduce(): retorna um novo array, de acordo com a condição informada;
forEach(): executa uma função para cada elemento de um array;
some(): verifica se pelo menos um elemento de um array atende à condição informada;
every(): verifica se todos os elementos de um array atendem à condição informada;
find(): retorna o primeiro elemento de um array que atende à condição informada;
findIndex(): retorna a posição do primeiro elemento de um array que atende à condição informada;
Math.random(): retorna um número aleatório entre 0 e 1;
Math.round(): arredonda um número para o inteiro mais próximo;
Math.ceil(): arredonda um número para o inteiro mais próximo, sempre para cima;
Math.floor(): arredonda um número para o inteiro mais próximo, sempre para baixo;
Math.pow(): retorna a potência de um número;
Math.sqrt(): retorna a raiz quadrada de um número;
Math.abs(): retorna o valor absoluto de um número;
Math.min(): retorna o menor valor de um conjunto de números;
Math.max(): retorna o maior valor de um conjunto de números;
Math.PI: retorna o valor de PI;
Math.E: retorna o valor de E;
Date(): retorna a data e a hora atual;
getFullYear(): retorna o ano atual;
getMonth(): retorna o mês atual;
getDate(): retorna o dia atual;
getDay(): retorna o dia da semana atual;
getHours(): retorna a hora atual;
getMinutes(): retorna o minuto atual;
getSeconds(): retorna o segundo atual;
getMilliseconds(): retorna o milissegundo atual;
getTime(): retorna o timestamp atual;
setFullYear(): altera o ano;
setMonth(): altera o mês;
setDate(): altera o dia;
setHours(): altera a hora;
setMinutes(): altera o minuto;
setSeconds(): altera o segundo;
setMilliseconds(): altera o milissegundo;
setTimeout(): executa uma função após um determinado tempo;
setInterval(): executa uma função em intervalos de tempo regulares;
clearTimeout(): cancela a execução de uma função agendada com o setTimeout();
clearInterval(): cancela a execução de uma função agendada com o setInterval().
addEventListener(): adiciona um evento a um elemento;
removeEventListener(): remove um evento de um elemento;
stopPropagation(): interrompe a propagação de um evento;
preventDefault(): cancela o comportamento padrão de um evento;
target: retorna o elemento que acionou o evento;
type: retorna o tipo de evento;
keyCode: retorna o código da tecla pressionada;
which: retorna o código da tecla pressionada;
charCode: retorna o código da tecla pressionada;
pageX: retorna a posição do eixo X do mouse;
pageY: retorna a posição do eixo Y do mouse;
offsetX: retorna a posição do eixo X do mouse, em relação ao elemento;
offsetY: retorna a posição do eixo Y do mouse, em relação ao elemento;
clientX: retorna a posição do eixo X do mouse, em relação à janela;
clientY: retorna a posição do eixo Y do mouse, em relação à janela;
screenX: retorna a posição do eixo X do mouse, em relação à tela;
screenY: retorna a posição do eixo Y do mouse, em relação à tela;
altKey: retorna true se a tecla ALT foi pressionada;
ctrlKey: retorna true se a tecla CTRL foi pressionada;
shiftKey: retorna true se a tecla SHIFT foi pressionada;
button: retorna o botão do mouse pressionado;
buttons: retorna os botões do mouse pressionados;
